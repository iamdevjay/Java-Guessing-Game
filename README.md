#  Java Number Guessing Game

A simple and interactive console-based game built in Java. Test your luck and logic by trying to guess a secret number between 1 and 100!

##  How to Play
1. Run the program and enter your name.
2. The computer will pick a random number between **1 and 100**.
3. You have **10 attempts** to guess it.
4. The game will tell you if your guess is **too high** or **too low** after every turn.

##  How to Run (For Players)
1. Download the `GuessingGameProject.jar` from the [Releases](https://github.com/iamdevjay/Java-Guessing-Game/releases) section.
2. Open your Command Prompt (Terminal).
3. Navigate to your Downloads folder using `cd Downloads`.
4. Run the game by typing:
   ```bash
   java -jar GuessingGameProject.jar
